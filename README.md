# AutomationTesterAssignment
Automation Tester Intern assignment: Selenium Java scripts

# Test Cases
# TC001_LoginTest
 •	Purpose: Tests the login functionality of the application.
 •	Scenarios Covered:
    o	Login with an empty email field.
    o	Login with an invalid email format.
    o	Login with a valid email and OTP.
    o	Login with an invalid OTP.
# TC002_CompleteProfile
 •	Purpose: Tests the profile completion functionality.
 •	Scenarios Covered:
    o	Submitting an empty profile form.
    o	Submitting a valid profile form.
# TC003_AddChild
 •	Purpose: Tests the functionality of adding a child.
 •	Scenarios Covered:
    o	Submitting an empty form for adding a child.
    o	Completing the form with valid details.

# Steps to execute the script:
Open the following test case files located in project directory:
src/test/java/testcases/TC001LoginTest.java
src/test/java/testcases/TC002CompleteProfile.java
src/test/java/testcases/TC003AddChild.java
Use IDE (such as IntelliJ IDEA or Eclipse) or a terminal to execute the test cases.

# Expected Outcomes
•	TC001_LoginTest: Validates login functionality with different email and OTP scenarios.
•	TC002_CompleteProfile: Ensures profile completion works correctly with valid and invalid inputs.
•	TC003_AddChild: Confirms that the child addition process functions as expected.
